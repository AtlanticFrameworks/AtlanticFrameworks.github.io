Kommunikations-Protokoll

Funk-Codes & Status

Offizielle Funkcodes der Bundeswehr Roleplay Community. Jedes Mitglied ist verpflichtet, diese Codes im aktiven Funkverkehr präzise anzuwenden.

4.1 Einsatz-Codes

C1
Code 1
Anfahrt ohne Siren ohne Blaulicht
Normale Anfahrt zum Einsatzort unter Einhaltung der StVO.
C2
Code 2
Anfahrt ohne Siren mit Blaulicht
Erhöhte Eile. Sonderrechte ohne akustische Warnsignale.
C3
Code 3
Anfahrt mit Sirene und Blaulicht
Höchste Eile. Volle Inanspruchnahme von Sonder- und Wegerechten.
C4
Code 4
Einsatzort beendet / alles in Ordnung
Lage gesichert, keine weiteren Maßnahmen erforderlich. [+ Ort]
C5
Code 5
Geheime Ermittlung / Außer Sichtweite
Observation oder verdeckte Operation. Unauffällig verhalten. [+ Ort]
C6
Code 6
Ankunft zu Fuß
Einheit hat den Einsatzort fußläufig erreicht.

NOTFALL-CODES

11-20

11-99
Unter Beschuss

20-20

Anforderung von zusätzlichen Einheiten zur Unterstützung. [+ Ort]

Prio-Funk (Priorität)

Wird bei dringenden Lagen genutzt (z. B. 10-80, 11-99). Sobald ein Prio-Funk ausgerufen wird, gilt für alle unbeteiligten Einheiten absolute Funkstille!

Befehl zur Funkstille:
"STOPFEN!"

Nach diesem Befehl darf nur noch die meldende Einheit und die Leitstelle sprechen.
